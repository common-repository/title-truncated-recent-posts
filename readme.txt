== Description ==

Title truncated recent posts widget, and you can also set the prefix or suffix of titles.

== Installation ==

1. Download and unzip the zip file into your Wordpress `plugins` folder such that the plugin files are at: `wp-content/plugins/title-truncated-recent-posts/...`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The new recent posts widget will show in the 'Widget' page.
